# COURSE_IMPORT_FORMAT.md — Edunancial Claude Course Import Specification

This document defines the standardized import format for importing course packages into the Edunancial platform, whether generated by Claude AI or authored manually.

Import URL: `/admin/courses/import`
Import API: `POST /api/admin/import`

---

## Overview

The Edunancial Course Import system accepts a **JSON course package** that describes a complete course including its metadata, modules, lessons, lesson content blocks, and assets.

Supported actions:
- `preview` — Validate and preview without saving
- `import` — Validate and save as a draft course

---

## Top-Level Course Object

| Field | Type | Required | Description |
|---|---|---|---|
| `title` | `string` | ✅ | Course title |
| `slug` | `string` | No | URL slug (auto-generated from title if omitted) |
| `subtitle` | `string` | No | One-line subtitle |
| `description` | `string` | No | Full course description |
| `path` | `"red" \| "white" \| "blue"` | ✅ | Learning path color |
| `language` | `"en" \| "es" \| "fr" \| "fr-CA"` | ✅ | Course language |
| `region` | `string \| null` | No | Target region (e.g. `"north-america"`) |
| `country` | `string \| null` | No | Target country code (e.g. `"US"`) |
| `difficulty` | `"beginner" \| "intermediate" \| "advanced"` | No | Default: `"beginner"` |
| `duration` | `string` | No | Estimated total duration (e.g. `"6 hours"`) |
| `membershipTier` | `"free" \| "basic" \| "premium" \| "elite"` | No | Default: `"free"` |
| `instructorName` | `string` | No | Instructor display name |
| `instructorBio` | `string` | No | Instructor bio |
| `thumbnailUrl` | `string \| null` | No | Thumbnail image URL |
| `tags` | `string[]` | No | Searchable tags |
| `categories` | `string[]` | No | Content categories |
| `prerequisites` | `string[]` | No | Prerequisite knowledge |
| `learningObjectives` | `string[]` | No | Learning objectives list |
| `modules` | `Module[]` | No | Course modules (see below) |
| `seo` | `SeoMeta` | No | SEO metadata (see below) |
| `relatedCourseIds` | `string[]` | No | IDs of related courses |
| `relatedMarketplaceIds` | `string[]` | No | IDs of related marketplace items |

### Path Colors

| Value | Meaning | Content Focus |
|---|---|---|
| `"red"` | RED Path | Real Estate investing |
| `"white"` | WHITE Path | Paper Assets (stocks, bonds, options) |
| `"blue"` | BLUE Path | Business and entrepreneurship |

## Module Object

| Field | Type | Required | Description |
|---|---|---|---|
| `title` | `string` | ✅ | Module title |
| `slug` | `string` | No | Auto-generated from title if omitted |
| `description` | `string` | No | Module overview |
| `order` | `number` | No | Display order (auto-assigned if omitted) |
| `lessons` | `Lesson[]` | No | Lessons in this module |

---

## Lesson Object

| Field | Type | Required | Description |
|---|---|---|---|
| `title` | `string` | ✅ | Lesson title |
| `slug` | `string` | No | Auto-generated from title if omitted |
| `description` | `string` | No | Lesson summary |
| `duration` | `string` | No | Estimated reading/watch time (e.g. `"15 min"`) |
| `order` | `number` | No | Display order (auto-assigned if omitted) |
| `content` | `string` | No | Shorthand: markdown text content (creates one text block) |
| `blocks` | `Block[]` | No | Full lesson content blocks (see below) |
| `aiCoachPrompt` | `string` | No | Prompt shown to the AI coach for this lesson |

> If both `content` (shorthand) and `blocks` are provided, `blocks` takes precedence.

---

## Lesson Block Object

Lesson blocks are the atomic content units within a lesson.

| Field | Type | Required | Description |
|---|---|---|---|
| `type` | `ContentType` | ✅ | Block content type (see below) |
| `title` | `string` | No | Block heading or label |
| `content` | `string` | No | Text/markdown content |
| `url` | `string \| null` | No | URL (video, audio, external link) |
| `mediaId` | `string \| null` | No | ID of an uploaded media item |
| `order` | `number` | No | Display order (auto-assigned if omitted) |

### Block Content Types

| Type | Description | `content` | `url` |
|---|---|---|---|
| `"text"` | Formatted text/markdown | Markdown text | — |
| `"video"` | YouTube embed or video link | Caption/description | YouTube URL or video URL |
| `"audio"` | Audio reference | Caption | Audio URL |
| `"pdf"` | PDF resource | Description | PDF URL or leave blank for upload |
| `"docx"` | Word document | Description | DOCX URL |
| `"image"` | Image | Alt text / caption | Image URL |
| `"quiz"` | Knowledge check | Quiz JSON or description | — |
| `"download"` | Downloadable file | File description | Download URL |
| `"external-link"` | External resource | Link description | Full URL |
| `"ai-coach"` | AI coach prompt | Prompt text | — |

## Import API

### Preview (validate without saving)

```http
POST /api/admin/import
Content-Type: application/json
x-csrf-token: <token from /api/admin/auth/session>

{
  "action": "preview",
  "payload": { ...CoursePackage... }
}
```

Response (success):
```json
{
  "ok": true,
  "preview": { ...AdminCourse... },
  "moduleCount": 3,
  "lessonCount": 12
}
```

Response (validation error):
```json
{
  "ok": false,
  "errors": ["course.title is required", "course.path must be 'red', 'white', or 'blue'"]
}
```

### Import (save as draft)

```http
POST /api/admin/import
Content-Type: application/json
x-csrf-token: <token from /api/admin/auth/session>

{
  "action": "import",
  "payload": { ...CoursePackage... }
}
```

Response (success):
```json
{
  "ok": true,
  "courseId": "course_abc123_xyz",
  "course": { ...AdminCourse... }
}
```

## Markdown Import (Flat Text)

For simpler content, Claude can also export a Markdown document that you paste into the JSON `content` field of a lesson block:

```markdown
## Lesson Title

This is lesson content in markdown format.

### Section 1

Regular paragraph text here.

- Bullet point 1
- Bullet point 2

### Key Takeaway

The most important thing to remember from this lesson.
```

This gets placed as a `"text"` block in the lesson blocks array.

## ZIP Package Import

For larger course packages, create a ZIP file with:

```
course-package.zip
├── course.json          ← The JSON course package (see above)
├── media/
│   ├── thumbnail.jpg
│   ├── lesson-1-worksheet.pdf
│   └── intro-video-reference.txt
└── README.md            ← Optional description
```

Upload the ZIP via `/admin/content/upload`. The system will extract and classify all files. Approve files via the batch review interface.

## Validation Rules

The import API enforces these rules:

| Rule | Error |
|---|---|
| `title` must be present | `course.title is required` |
| `path` must be `"red"`, `"white"`, or `"blue"` | `course.path must be 'red', 'white', or 'blue'` |
| `language` must be present | `course.language is required` |
| `modules` must be an array if present | `modules must be an array` |
| Each module must have `title` | `modules[N].title is required` |
| Module `lessons` must be array if present | `modules[N].lessons must be an array` |

## Preview Before Publish

Always use `action: "preview"` before `action: "import"` to validate:

1. POST `{ action: "preview", payload: {...} }`
2. Review the preview response — check module count, lesson count, path, language
3. If validation passes, POST `{ action: "import", payload: {...} }`
4. You are redirected to the course editor where you can further edit before publishing

---

## Rollback / Cancellation

- Imported courses are always saved as `"draft"` status
- Draft courses are never visible to students
- To cancel an import, simply delete the draft course from `/admin/courses`
- You can edit any field after import before publishing

## Claude Prompt Template

When asking Claude to generate a course package, use this prompt:

```
Generate a complete Edunancial course package in JSON format following the COURSE_IMPORT_FORMAT specification.

Course requirements:
- Title: [COURSE TITLE]
- Learning Path: [red/white/blue]
- Language: [en/es/fr]
- Difficulty: [beginner/intermediate/advanced]
- Membership Tier: [free/basic/premium/elite]
- Target audience: [DESCRIBE AUDIENCE]
- Number of modules: [N]
- Lessons per module: [N]

Content requirements:
- Each lesson should include a text block with educational content
- Include at least one AI coach prompt per lesson
- Include learning objectives at the course level
- Include prerequisites
- Follow Edunancial's educational standards: emphasize practical application, no income guarantees

Output format: Valid JSON only, matching the Edunancial Course Import Format specification.
```

## Course Status Workflow

After import, courses follow this workflow:

```
draft → published → archived
               ↑
           unpublish (→ draft)
```

- `draft` — Not visible to students, editable
- `published` — Visible to students with the correct membership tier
- `archived` — Hidden from all views, read-only

Only `admin` and `content_manager` roles can change course status.
