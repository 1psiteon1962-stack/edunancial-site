Compromiso

Edunancial, Inc. se compromete a que su sitio web y contenido en línea sean lo más accesibles posible, incluyendo para personas con discapacidades. Procuramos alinearnos con estándares reconocidos de accesibilidad web (WCAG).

Medidas

  • Uso de encabezados claros y HTML estructurado;
  • Textos alternativos para imágenes clave;
  • Fuentes legibles y contraste de color adecuado;
  • Diseños pensados para navegación por teclado.

Limitaciones

Aunque trabajamos para que todo sea accesible, cierto contenido antiguo, herramientas de terceros o videos incrustados pueden no cumplir aún todos los estándares. Continuamos mejorando estas áreas.
